## Descrição do caso escolhido 
O projeto foi desenvolvido para um serviço educacional fictício chamado Foco Educação, inspirado em professores particulares e plataformas de reforço escolar existentes em Brasília. 

O Foco Educação oferece aulas de reforço para estudantes do Ensino Médio e preparação especializada para ENEM e vestibulares, abrangendo disciplinas como Matemática, Física, Química e Redação. 

Atualmente, muitos professores e pequenos projetos educacionais utilizam apenas redes sociais para divulgação, o que dificulta a organização das informações, apresentação dos serviços e contato com alunos interessados. 

Por esse motivo, foi proposta a criação de um site institucional responsivo para apresentar os serviços, metodologia de ensino, disciplinas disponíveis e facilitar o agendamento de aulas. 

## Necessidade identificada 
O projeto foi escolhido por representar uma necessidade real e comum no cenário educacional atual. 

Muitos estudantes têm dificuldades em disciplinas específicas e buscam reforço especializado para melhorar seu desempenho escolar e aumentar suas chances de aprovação em vestibulares. 

Esse caso também permite aplicar os conteúdos estudados durante a disciplina: 

• Estruturação semântica em HTML 

• Estilização com CSS 

• Responsividade 

• Usabilidade 

• Interatividade com JavaScript 

• Uso de Bootstrap 

Além disso, o tema apresenta grande relevância social, pois está diretamente ligado ao desenvolvimento acadêmico. 

## Público-alvo 
O público-alvo é composto por: 

• Estudantes do Ensino Médio 

• Vestibulandos 

• Candidatos ao ENEM 

• Alunos com dificuldades em matérias exatas 

• Pais que buscam reforço escolar para seus filhos

Espera-se que o visitante encontre as seguintes informações na página: 

• Disciplinas oferecidas 

• Metodologia de ensino 

• Formato das aulas (online/presencial) 

• Benefícios do acompanhamento 

• Simulados e materiais 

• Formas de contato 

• Perguntas frequentes

## Principais decisões do projeto 
Foi escolhida uma identidade visual associada ao ambiente acadêmico. 

Paleta de cores: 

• Azul escuro 

• Branco 

• Amarelo 

• Cinza claro 

Tipografia: Fonte limpa e moderna para facilitar a leitura. 

Layout: Foi adotado um layout profissional e intuitivo. 

A organização visual segue uma sequência lógica: 

1.	Apresentação
   
2.	Serviços
   
3.	Disciplinas
   
4.	ENEM
   
5.	FAQ
    
6.	Contato
    

## Funcionalidade em JavaScript 

Foi implementada uma seção de Perguntas Frequentes (FAQ). 

Funcionamento: Ao clicar na pergunta, a resposta é exibida ou ocultada. 

Objetivo: Facilitar a consulta rápida de dúvidas comuns.

Benefício: Melhora a experiência do usuário e reduz dúvidas repetitivas.
